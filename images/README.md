# Images Folder

This folder contains images for the website.

## Required Images

### Team Photo
- **Filename**: `team-photo.jpg`
- **Description**: Official group photo of the BEQ team
- **Recommended size**: 1800px wide, high quality
- **Format**: JPG or PNG

### Individual Member Photos (Optional)
Add individual photos for team members with filenames like:
- `marcel-vieira-lara.jpg`
- `marc-pedersen.jpg`
- `marieke-warmerdam.jpg`
- etc.

**Recommended specs for member photos**:
- Size: 400x400px or larger (square format)
- Format: JPG or PNG
- File naming: lowercase with hyphens (e.g., `first-last.jpg`)

## Usage

To add the team photo, replace the placeholder in `index.html` (search for "team-photo-placeholder").

To add individual photos, replace the `<div class="photo-placeholder">` elements in the team member cards.

See the README.md file for detailed instructions.
